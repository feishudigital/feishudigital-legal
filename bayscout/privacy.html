Privacy Policy

BayScout — eBay Seller Tools Chrome Extension — Last updated: February 2026

Your browsing data stays private. BayScout reads eBay page data locally in your browser to provide analysis. We only store data you explicitly choose to save (tracked items, alerts, keyword lists).

What data we collect

BayScout collects only what is necessary to provide its features:

  • Account data: Email address and hashed password when you create an account
  • Saved items: Items, sellers, and keywords you choose to track
  • Usage count: Daily API call count for free-tier limits
  • Analysis history: Search and listing analyses you run (for your dashboard)

We do not collect browsing history, eBay credentials, personal financial data, or any data from non-eBay websites.

How the extension works

BayScout operates in two modes:

  • Client-side analysis: Search stats, listing scores, title analysis, and profit calculations are computed entirely in your browser. No eBay page content is sent to our servers for these features.
  • Backend features: Keyword research, price tracking, seller monitoring, and alerts require server communication. Only the specific data needed (item IDs, keywords, seller names) is transmitted — never full page content.

Network requests

The extension communicates with:

  • BayScout API (our backend) — for authentication, keyword research, price tracking, seller tracking, alerts, and AI listing optimization
  • eBay Autocomplete API — for keyword suggestions (public API, no personal data sent)
  • Lemon Squeezy API — for Pro subscription license validation only

We do not use any analytics, tracking pixels, or third-party advertising services.

Data storage

Locally (your browser):

  • Authentication tokens (via chrome.storage.local)
  • User preferences and settings
  • Daily usage counter

Server-side (Supabase, hosted in EU/US):

  • Your account profile and subscription status
  • Tracked items, sellers, and price history
  • Keyword lists and alert configurations
  • Analysis history

All server data is protected by Row Level Security — you can only access your own data. Data is encrypted in transit (TLS) and at rest.

AI features

BayScout's AI Listing Tools may use third-party AI providers (OpenAI, Anthropic) to generate optimized listing titles and descriptions. When you use AI features:

  • Only the listing title, description, and category you provide are sent to the AI provider
  • No personal data or browsing history is included
  • AI-generated content is returned to you and not stored by us beyond your analysis history

If no AI API is configured, a rule-based fallback is used that runs entirely on our server with no third-party data sharing.

Permissions explained

  • Host permissions (ebay.com/co.uk/de/com.au/fr): To read listing data and inject analysis overlays on eBay pages only
  • storage: To save authentication tokens and preferences locally
  • tabs: To open the BayScout dashboard in a new tab

Data deletion

You can delete your account and all associated data at any time by contacting us at feishudigital@gmail.com. Uninstalling the extension removes all local data immediately.

Third-party services

  • Supabase (database & authentication) — supabase.com/privacy
  • Lemon Squeezy (payments) — lemonsqueezy.com/privacy
  • OpenAI / Anthropic (optional AI features) — openai.com/privacy, anthropic.com/privacy

Children's privacy

BayScout is not intended for use by children under 13. We do not knowingly collect data from children.

Changes to this policy

If we change this policy, the updated version will be posted here and noted in the Chrome Web Store listing. Material changes will be communicated via the extension.
