2. Service description

Readability Scorer is a Chrome browser extension that provides real-time readability analysis for text fields on any website. It computes readability scores (Flesch-Kincaid, Gunning Fog, Coleman-Liau, SMOG, ARI, Dale-Chall) and detects writing issues (passive voice, adverbs, complex words, clichés, hedge words, sentence repetition). It also offers inline highlighting of issues via the CSS Custom Highlight API, writing goals with daily word count and streak tracking, and sentence-level analysis. All analysis is performed entirely in your browser. We do not operate a backend server and do not have access to your text or data.
