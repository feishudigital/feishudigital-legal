Readability Scorer — Chrome Extension — Last updated: February 12, 2026

2. What data we process

All data is stored locally in your browser (chrome.storage.local) and never transmitted to our servers:

  • Preferences: Theme, panel border, panel opacity, target score, auto-analyze toggle, badge visibility
  • Usage tracking: Daily analysis count and last reset date (for free tier limit of 5 analyses per day)
  • License data: Pro license key, license instance ID, and last validation timestamp (if purchased)

We do not store, read, or transmit the text you type or analyze. All readability formulas (Flesch-Kincaid, Gunning Fog, Coleman-Liau, SMOG, ARI, Dale-Chall) are pure mathematics computed entirely on your device. Text is processed in memory and immediately discarded.

3. How the extension works

The extension runs as a content script on all websites to detect text fields (textarea, contenteditable elements). When you type or focus a text field, the extension:

  • Extracts the text content from the active text field
  • Computes readability scores and writing issue detection entirely in your browser
  • Displays results in a floating badge and detail panel (rendered in a Shadow DOM overlay)

At no point is your text content sent to any server, logged, or stored persistently. The analysis happens in real-time in browser memory.

4. Network requests and third-party data processing

The extension makes zero network requests during normal use. The only exception:

  • Lemon Squeezy API (api.lemonsqueezy.com) — to activate, validate, and deactivate Pro license keys. Only the license key string and instance identifier are transmitted. No text content, browsing data, personal information, or usage statistics are sent.

Lemon Squeezy (Lemon Squeezy, LLC) processes payment data when you purchase a Pro subscription. This includes your name, email, and payment information. Lemon Squeezy acts as Merchant of Record and is an independent data controller for payment data. See: Lemon Squeezy Privacy Policy.

We do not use analytics, tracking pixels, telemetry, cookies, or any third-party advertising services. There is no server-side backend.

5. Data transfer to third countries

License key validation requests are sent to Lemon Squeezy's servers, which may be located in the United States. This transfer is necessary for the performance of the license agreement (Art. 49(1)(b) GDPR). No personal data beyond the license key string and instance identifier is transmitted by the extension.

6. Legal basis for processing (Art. 6 GDPR)

  • Local data storage (preferences, usage count): Art. 6(1)(b) GDPR — necessary for the performance of the service you requested by installing the extension.
  • License key validation: Art. 6(1)(b) GDPR — necessary for the performance of the Pro subscription contract.
  • Payment processing by Lemon Squeezy: Art. 6(1)(b) GDPR — necessary for the purchase transaction. Lemon Squeezy processes this data as an independent controller.

7. Data retention

  • Local browser data: Retained until you uninstall the extension or clear extension data. No time limit — you control deletion.
  • Text content: Not retained at all. Processed in memory during analysis and immediately discarded.
  • Payment data at Lemon Squeezy: Retained according to Lemon Squeezy's own retention policy and applicable tax/accounting laws.

8. Permissions explained

  • Content script on all URLs (<all_urls>): The extension needs to detect and analyze text fields on any website you visit. The content script only activates when it finds a text input field. It does not read page content outside of text fields.
  • activeTab: To access the currently active tab for text field detection.
  • storage: To save your preferences, usage count, and license key locally in your browser.
  • alarms: To periodically revalidate Pro license status. This triggers a single API call to Lemon Squeezy containing only the license key.

Under the General Data Protection Regulation (GDPR), you have the following rights:

  • Right of access (Art. 15) — You can request information about what data we process. Since all data is stored locally in your browser, you already have full access via the extension's options page.

  • Right to erasure (Art. 17) — Uninstall the extension or clear its data to delete all locally stored data. For payment data, contact Lemon Squeezy.
